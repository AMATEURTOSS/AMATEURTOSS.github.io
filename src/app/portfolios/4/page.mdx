# 브릿지 프로젝트

[서비스 URL](https://briiidgeproject.com)

온라인으로 대치동의 공부 컨설팅을 제공하는 서비스입니다.

- 학생의 공부습관, 평소 성적, 학구열 등으로 진단을 받고 앞으로 예상 성적, 예상 대학 등의 정보를 확인할 수 있습니다.
- 진단 내용을 바탕으로 컨설팅을 받을 수 있습니다.
- 매주 컨설팅을 받고 공부계획을 받을 수 있습니다.
- 공부계획대로 공부를 하고 사진을 업로드하여 공부내용을 인증할 수 있습니다.

## 기술 스택

- BE: NodeJS, Typescript, ExpressJS, MongoDB 등…
- FE: React, React Native, Typescript, NextJS, MUI, Recoil, SWR 등…

## 담당 업무

- 유저 사이드 클라이언트
  - 공부달력 기능 개발
  - 공부달력 퍼포먼스 개선 (불필요한 리렌더링 제거 800ms → 80ms로 속도 개선)
  - 리팩토링, 불필요한 useEffect, state 등 제거 및 컴포넌트 분리
- 어드민 사이드 클라이언트
  - 결제 목록, B2B 결제 목록, 결제 예약 목록 페이지 개발
- 서버
  - 결제 서버 유지보수
  - B2B 결제 로직 개발 (단건, 구독)
  - 학부모 회원이 학생의 화면을 볼 수 있게 하는 `자녀 연결` 기능 개발
